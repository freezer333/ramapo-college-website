Lab 08 - Extending the Linked List

Download the LinkedList.cpp file from appiversity.  For this lab, add a new public function called addInOrder to the linked list class.  

In main, write your own code which creates a LinkedList object and adds (using only your new addInOrder function) 10 random numbers between 1 and 100.  Print out the linked list to test its effectiveness.

Note:  You should not call any of the other add functions from main - and your addInOrder function need not work correctly if someone mixes addInOrder with calls to the other add functions.

Side note:  This isn’t a very good design… we shouldn’t have “unwritten” rules about how to use a class.  A better solution would be to create a SortedList class that simply doesn’t have the other add functions… you’d be wise to keep this in mind :)
